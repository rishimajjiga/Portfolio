It means AI isn't just used to write the code faster — it's a core feature inside the product. In MediFlow, AI analyzes symptoms. In LillyRecipes, AI generates personalized recipes. In FitnessPro, AI adapts workout plans. I spend significant time on prompt engineering, API integration, and making the AI feel natural and useful to end users.

What technologies do you specialize in?

Full-stack: HTML, CSS, JavaScript, Python, Flask, Firebase. AI: OpenAI API, Gemini API, prompt engineering. I also work with real-time databases, authentication systems, and third-party API integrations.

Are you open to freelance or team collaborations?

Absolutely. I'm open to freelance projects, startup collaborations, and full-time opportunities. Whether you need a complete product built or AI features integrated into an existing app, reach out!

Do you build full-stack applications end-to-end?

Yes — I handle frontend design, backend APIs, database architecture, and AI/third-party integrations. I can take a product from wireframe to live deployment.
